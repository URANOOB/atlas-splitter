# Manifiestos

**Síntoma:** Error validando manifiesto al aplicar revisión.
**Causa probable:** Editaste el JSON manualmente rompiendo la sintaxis.
**Comprobación:** Abre el archivo JSON en un editor de código y busca marcas rojas indicativas de error.
**Solución:** Corrige la sintaxis (comas faltantes, comillas erróneas) y vuelve a intentarlo.
**Comando de diagnóstico:** 
```text
atlas-splitter review resultados/atlas
```
