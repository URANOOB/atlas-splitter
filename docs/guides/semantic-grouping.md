# Agrupación semántica

## Requisitos

Dependencias AI y Qwen3-VL ya descargado explícitamente.

```text
atlas-splitter setup ai
atlas-splitter models download qwen3-vl-2b
atlas-splitter semantic atlas.webp --output resultados
```

Este modo separa, agrupa, crea `review.json` y actualiza el reporte. Los nombres son inferencias locales: revísalos antes de usarlos.
