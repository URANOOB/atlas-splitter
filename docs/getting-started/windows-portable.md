# Windows portable

Descomprime `AtlasSplitter-Lite.zip`, abre `README-WINDOWS.txt` y ejecuta:

```powershell
.\AtlasSplitter-Lite.exe doctor
.\AtlasSplitter-Lite.exe split atlas.webp --output resultados
```

`VERSION.txt`, `LICENSE` y `SHA256SUMS.txt` permiten identificar y verificar la entrega. Lite no incluye modelos ni CUDA.
