# Referencia CLI

La referencia se obtiene de la CLI instalada:

```text
atlas-splitter --help
atlas-splitter split --help
atlas-splitter extract --help
```

Comandos principales: `split`, `extract`, `semantic`, `group`, `group-3d`, `review`, `preview`, `inspect`, `doctor`, `setup` y `models`. Los alias `run`, `glb`, `semantic-3d` e `install` siguen disponibles temporalmente para compatibilidad.
